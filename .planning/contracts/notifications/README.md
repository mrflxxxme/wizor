# Context: `notifications` — Alerts & Notifications

**Purpose:** Отправка алертов деградации (падение Visibility/Citation-метрик вне полосы шума) и уведомлений о событиях платформы по трём каналам: email (Mailgun / Unisender), Telegram, webhook. Алерт отправляется, только если метрика вышла за полосу шума (N≥5+CI) — не при внутришумовых флуктуациях. Конфигурируется пользователем (какие события, какой канал).

**Owns (data):** Конфигурации каналов уведомлений (endpoint, credentials — зашифровано), шаблоны сообщений, история отправок (для дедупликации и аудита).
